Lista med data till HTML-element

  • Hur kan vi omvandla vår lista med djur som strängar till html-element?
const olikaDjur = [ "Jag ser en katt", "Jag ser en hund", "Jag ser en fågel" ];

Ska omvandlas till:

const minaParagrafElement = [
        <p>Jag ser en katt</p>,
        <p>Jag ser en hund</p>,
        <p>Jag ser en fågel</p>
    ]

Vi använder map

  • Med map kan vi köra en funktion för varje element i en array
  • Resultatet av varje funktion blir ett objekt i en ny array
const olikaDjur = [
    "Jag ser en katt",
    "Jag ser en hund",
    "Jag ser en fågel"
];
const minaParagrafElement = olikaDjur.map(djur => <p>{djur}</p>)

Nu är komponenten klar

const OlikaDjur = () => {
    const olikaDjur = [
        "Jag ser en katt",
        "Jag ser en hund",
        "Jag ser en fågel"
    ];
    const minaParagrafElement = olikaDjur.map(djur => <p>{djur}</p>)
    return <>{minaParagrafElement}</>;
}
export default OlikaDjur;

Unik nyckel - Key

  • React måste ha en unik nyckel för att hålla reda på varje element i en lista
  • Man lägger till ett key-attribut på varje html-element som listas upp
  • Värdet på nyckeln är vanligtvis ett id som varje objekt har från api:et
  • Det är viktigt att värdet på nyckeln är samma hela tiden för ett visst objekt

Varning om man inte har nyckel

  • Koden fungerar utan nyckel men en varning visas i consolen:

Key-attribut - exempel

const olikaDjur = [
    { id: 1, text: "Jag ser en katt" },
    { id: 2, text: "Jag ser en hund" },
    { id: 3, text: "Jag ser en fågel" }
];
const minaParagrafElement = olikaDjur.map(djur => <p key={djur.id}>{djur.text}</p>)

Övning - lägga till key

  • Utgå ifrån förra övningen
  • Byt ut animals-listan mot nedanstående med unika nycklar (id)
  • Lägg till key-attribut så att varningen i consolen försvinner
