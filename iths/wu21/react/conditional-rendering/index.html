Conditional Rendering

Jonatan Hallenberg

Conditional rendering

  • Ganska ofta har man två olika komponenter som man vill rendera på samma ställe beroende på ett kriterium
  • Och ibland vill man bara rendera en komponent om ett kriterium är uppfyllt och annars inte rendera någonting
  • Vi använder vanlig JavaScript-kod som if-sats, ternary operator och &&-operatorn för att lösa detta

Exempel - if-sats

const Form = () => (
    <form>
        <input type="text" placeholder="e-post" />
        <button type="submit">Skicka</button>
    </form>)

const ConfirmationText = () => <p>Tack! Vi hör av oss inom kort.</p>;

const ContactForm = ({isSent}) => {
    if (isSent) {
        return <ConfirmationText />
    } else {
        return <Form />
    }
}

Ternary operator

  • För att förenkla koden kan vi skriva om if-satsen som en ternary-operator:
condition ? true : false;
  • Och vi kan då skriva allt på en rad:
const ContactForm = ({isSent}) => isSent ? <ConfirmationText /> : <Form />

&&-operatorn

  • För att rendera en komponent endast om ett kriterium är uppfyllt kan vi använda &&-operatorn
  • En CSS-class som finns här kan användas var som helst på sidan
const CartText = ({ products }) => (
    <h2>Varukorg</h2>
    { products.length && <p>Du har {products.length} produkter i varukorgen.</p>}
)
